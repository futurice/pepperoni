WHAT IS PEPPERONI?

Delicious framework for mobile development

A blueprint for building cross platform mobile experiences rapily using ready-to-use integrated building blocks for common mobile app features.

Under the hood the startup kit is crafted on a solid foundation with a modern architecture and industry best practices, featuring:

  • Native mobile apps built with React Native by Facebook and Redux

  • Mobile backend powered by NodeJS and PostgreSQL - Coming soon

  • Cloud-ready for Heroku, Amazon Web Services, Google Cloud Platform, Microsoft Azure, etc - or host it yourself!
